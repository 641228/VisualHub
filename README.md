# VisualHub
## English
### Introduction
VisualHub is an open-source front-end template for image hosting. It includes full user system and image management functions, compatible with desktop and mobile devices.
### Features
- User registration, login and logout
- Support click and drag to upload images
- Image download and link copy
- Fully responsive layout
- Clean code structure, easy for secondary development
### Usage
1. Download the `index.html` file.
2. Open the file with a text editor.
3. Find the code: `const API_BASE = "http://localhost:8080/api"`.
4. Replace the address with your backend API URL.
5. Save the file, then open it in a browser or deploy to your web server.
## 中文
### 简介
VisualHub 是一款开源图床前端模板，内置完整账号系统与图片管理功能，适配电脑与移动设备。
### 功能
- 用户注册、登录、退出
- 支持点击、拖拽两种图片上传方式
- 图片下载、链接一键复制
- 全尺寸自适应布局
- 代码结构整洁，方便二次开发
### 使用方法
1. 下载 `index.html` 文件。
2. 使用文本编辑器打开文件。
3. 找到代码行：`const API_BASE = "http://localhost:8080/api"`。
4. 将地址替换为你的后端接口地址。
5. 保存文件，直接使用浏览器打开，或部署至网页服务器即可。
